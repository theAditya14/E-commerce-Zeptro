# 🤝 Contributing

Thank you for your interest in contributing to **Zaptro E-Commerce**! 🚀

This project is built using modern frontend technologies and is open source for anyone who wants to learn, contribute, or improve it.

## 🛠 Tech Stack

* ⚛️ React.js
* 🎨 Tailwind CSS
* 🛣 React Router
* 🔐 Clerk Authentication
* 📦 DummyJSON API
* 🌐 Vercel (Deployment)

# 🐞 Current Limitations

The following features are still being worked on:

* ❌ The website is **not fully mobile responsive** yet.
* ❌ Shopping cart data is **not persisted using LocalStorage** yet.
* ❌ Some UI components still need improvements.
* ❌ Accessibility improvements are pending.
* ❌ Performance optimization is still in progress.

# 🚀 How to Contribute

1. Fork this repository.
2. Clone your fork.
3. Create a new branch.

```bash
git checkout -b feature/your-feature-name
```

4. Make your changes.
5. Test everything properly.
6. Commit your changes.

```bash
git commit -m "feat: describe your changes"
```

7. Push your branch.

```bash
git push origin feature/your-feature-name
```

8. Open a Pull Request.
